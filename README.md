## 1、脚手架安装

### npm
```
npm create npm-package@latest my-first-blog
``` 

### yarn
```
yarn create npm-package@latest my-first-blog
```

###  pnpm
```
pnpm create npm-package@latest my-first-blog
```

###  bun
```
bunx create-npm-package@latest my-first-blog --bun
```

## 2、打开项目，安装依赖

### npm
```
npm install
``` 

### yarn
```
yarn install
```

###  pnpm(推荐)
```
pnpm install
```

###  bun
```
bun install
```
